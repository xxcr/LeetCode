## 题目地址

https://leetcode-cn.com/problems/climbing-stairs/

## 题目描述

```
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
```

## 解答

### 方法：动态规划

入门级动态规划问题

#### 思路

**我们按照解题思路走一遍👇**

1. **第一步：状态定义**

   dp[i] 表示的含义:到第i阶方案数

2. **第二步： 确定状态转移方程**

   根据实际的情况，我们很容易想到👇

   到第i阶梯有两种方式

   - 第一种, 从i-1向上走一步即可

   - 第二中，从i-2向上走二步即可

   所以状态转移方程：

   ```js 
   dp[i] = dp[i-1] + dp[i-2]
   ```

    

3. **第三步,初始化状态，dp数组**

   ```js
   dp[1] = 1,dp[2] = 2
   ```

按照这个三步走的话，我们就可以写出完整的解题代码



#### 代码

```js
/**
 * @param {number} n
 * @return {number}
 */

var climbStairs = function(n) {
    let dp = []
    dp[1] = 1
    dp[2] = 2
    for (let i = 3; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2]
    }
    return dp[n]
}

```
